# m4s2mp4
> m4s 转 mp4 合并工具

[English version](README.en.md)

---

这是一个用于 macOS/Linux 终端的自动化脚本。一键去除 B 站 m4s 缓存文件的伪装头（9 字节的 0），并无损合并为标准的高清 MP4 视频。

## 📁 依赖与目录结构

使用前，请确保主目录下的文件结构如下：

```text
你的工作目录/
 ├── ffmpeg            # FFmpeg 可执行文件（必须存在）
 ├── m4s2mp4.sh        # 本合并脚本
 ├── 1201093266/       # B站缓存文件夹（数字命名）
 │   ├── xxxx_1.m4s    # 视频流文件
 │   └── xxxx_2.m4s    # 音频流文件
 ├── 168467103/        # 其他缓存文件夹...
```
